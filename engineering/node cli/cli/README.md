设计一个富交互脚手架。



需求描述：

设计一个脚手架CLI。用户可以选择多种工程模板（如：SPA工程、NodeJS工程等），并根据输入的参数（如：工程名、作者名等）渲染模板，将工程生成在当前目录。

要求整个使用的过程中交互和log信息友好而丰富。
